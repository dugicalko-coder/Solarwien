# SolarWien Landingpage

Professionelle Landingpage für SolarWien - Österreichs bestbewertetes Solarunternehmen.

## Deployment auf Netlify

1. Exportieren Sie dieses Projekt aus Figma Make
2. Gehen Sie zu [app.netlify.com/drop](https://app.netlify.com/drop)
3. Ziehen Sie den kompletten Projekt-Ordner auf die Webseite
4. Fertig!

## Build-Einstellungen

Wenn die Plattform nach Build-Einstellungen fragt:

- **Build Command:** `npm run build`
- **Output Directory:** `dist`
- **Install Command:** `npm install`

## Domain verbinden

Nach dem erfolgreichen Deployment können Sie Ihre Domain solarwien.com verbinden:

1. Gehen Sie zu den Domain-Einstellungen
2. Fügen Sie solarwien.com hinzu
3. Konfigurieren Sie die DNS-Einträge bei Ihrem Domain-Registrar
